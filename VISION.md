# RideMate (ڕێ) — Product Vision

## What is this?
A Kurdish carpooling app connecting Erbil, Sulaymaniyah, and Duhok. Not a taxi app. Not a business. A community tool where friends give friends rides and get a coffee in return.

## Tone & Personality
- Mixed Kurdish dialects — Hawleri, Sulaimani, and Badini expressions all welcome. The language is natural and reflects how people actually talk across Kurdistan.
- Never corporate, never formal, never "terms and conditions" energy
- Warm, human, a bit funny
- Every screen is an opportunity to connect — empty states, loading, errors, confirmations
- Cultural references welcome
- One shot to impress each user. Make it count.

## Core Philosophy
- Friction should come at the moment of highest motivation, not at the door
- Everyone enters as a passenger. Zero friction. See rides immediately.
- Verification only when someone wants to post a ride — at that point THEY need US
- The app serves the user, not the other way around

## User Flow
1. **Landing** → tells the story, gets you excited
2. **Google sign in** → one tap, you're in
3. **Home** → real rides on screen immediately
4. **Want to ride?** → tap, request, get approved, WhatsApp the driver
5. **Want to drive?** → upload docs, get verified, start posting

## Go-to-Market
- Controlled launch with 10-15 pre-verified driver friends
- Seed real rides so new users always see activity
- Wait for 2-3 seats booked before initiating rides
- Slowly accept and vet new drivers
- Never launch to an empty marketplace

## Technical Principles
- Everything built from first principles. If it doesn't contribute to core functionality, it doesn't exist.
- Less code = fewer bugs = faster app. Only keep what earns its place.
- The app relies on sentimental expressions through language rather than graphics, themes, and animations. Communication with the user is our core skill.
- Mobile-first, 480px max width
- RTL Kurdish with Noto Sans Arabic
- Orange (#df6530) primary color
- After all features done: first-principles code audit, delete everything non-essential
